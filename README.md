Movie List APP

A Flutter application for searching and browsing movie information using the TMDb API.

Features:
• Search movies by name
• Browse trending movies
• View detailed movie information (title, year, plot, director)
• Google Sign-In authentication (Firebase)
• Add movies to watched list
• Add movies to watchlist
• Custom user rating with stars
• Personal notes/comments for each movie
• Persistent local storage (SharedPreferences)
• Smooth UI with loading indicators

Technologies Used:
• Flutter
• Dart
• Firebase Authentication (Google Sign-In)
• TMDb API
• Dio (HTTP client)
• GetIt (Service Locator)
• SharedPreferences
• Flutter Spinkit
